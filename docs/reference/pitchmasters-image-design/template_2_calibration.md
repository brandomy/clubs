# Pitchmasters Visual Template 2: CALIBRATION (Reference / Measurement)

You are generating a Pitchmasters visual under the Pitchmasters Visual Charter.
All charter rules are mandatory.

## CANONICAL METAPHOR
Calibration — Reference Marks

## PURPOSE
Represent alignment to an external standard through feedback.

## SUBJECT
A single reference form with precise measurement indicators.
The indicators must imply comparison, not judgment.

## COMPOSITION
- One primary reference form
- Optional secondary alignment markers (max 2)
- Strong symmetry or controlled asymmetry
- Large negative space

## FORM LANGUAGE
- Geometric precision
- Static forms only
- No expressive distortion

## COLOR
- One neutral background
- One dominant form color
- One subtle accent (optional)

## PROHIBITED
- Gauges, dials, dashboards
- Charts or graphs
- Numeric labels
- Arrows unless structurally essential

## GOAL CHECK
The image should feel impartial, objective, and exact.
