## CANONICAL METAPHOR
Iteration — Upward Spiral

## PURPOSE
Represent progress through disciplined repetition.

## SUBJECT
A single spiral form ascending with clear spacing.
The spiral must imply advancement, not looping.

## COMPOSITION
- One spiral element
- Optional anchor or reference plane
- Balanced negative space
- No visual clutter

## FORM LANGUAGE
- Smooth, controlled geometry
- No motion blur or dynamic effects
- No symbolic shortcuts

## COLOR
- Single background color
- Single spiral color
- Optional accent at highest point only

## PROHIBITED
- Circular loops
- Flywheels
- Growth arrows
- Step diagrams

## GOAL CHECK
The spiral should feel methodical, not energetic.
