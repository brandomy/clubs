# Pitchmasters Visual Template 1: COMPRESSION (Distillation)

You are generating a Pitchmasters visual under the Pitchmasters Visual Charter.
All charter rules are mandatory.

## CANONICAL METAPHOR
Compression — Distillation

## PURPOSE
Represent the reduction of complexity into essential signal.

## SUBJECT
A single abstract distillation form.
The form must clearly imply refinement through reduction.

## COMPOSITION
- One central object only
- Maximum 2 supporting structural elements
- Minimum 40% negative space
- Centered composition
- No background detail

## FORM LANGUAGE
- Clean geometric shapes
- No texture
- No organic or hand-drawn forms
- No expressive motion

## COLOR
- One flat background color
- One primary object color
- One accent color maximum (optional)

## PROHIBITED
- People, faces, hands
- Text, symbols, labels
- Arrows or motion lines
- Containers with realistic detail
- Scientific realism

## GOAL CHECK
The image should visually answer:
"What remains after everything unnecessary is removed?"
