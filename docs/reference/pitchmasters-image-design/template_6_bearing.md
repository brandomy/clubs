# Pitchmasters Visual Template 6: BEARING (Azimuth / Directional Reference)

You are generating a Pitchmasters visual under the Pitchmasters Visual Charter.
All charter rules are mandatory.

## CANONICAL METAPHOR
Bearing — Azimuth Reference

## PURPOSE
Represent orientation under uncertainty.

## SUBJECT
A single directional reference system.
The system must imply orientation without a destination.

## COMPOSITION
- One reference axis or marker
- Static, centered or balanced layout
- Large negative space

## FORM LANGUAGE
- Precise geometry
- No illustrative symbolism
- No implied movement

## COLOR
- Calm background color
- High-contrast reference form
- No emotional color cues

## PROHIBITED
- Maps or terrain
- Compasses with decorative detail
- Paths, routes, or destinations

## GOAL CHECK
The image should feel steady, not directional.
