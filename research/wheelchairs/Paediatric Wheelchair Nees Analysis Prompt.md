# ASEAN Pediatric Wheelchair Crisis Research

## Objective

Identify where HuaQiao Foundation can have maximum impact distributing pediatric wheelchairs across ASEAN. We need to know:

1. **Which age cohort is most neglected**: 2-5, 6-12, or 13-18 years?
2. **Which countries have the greatest unmet need?**
3. **Where can one container (~500 wheelchairs, ~$24,000) create maximum impact?**

## Context

HuaQiao Foundation imports wheelchairs from China at $47.50 landed cost versus $380 retail (8× value multiplier). Research in Penang, Malaysia revealed 1,200 children waiting 8-18 months for wheelchairs—cost equals 92% of low-income family monthly income.

**Why this matters**: Ages 2-5 represent 90% of brain development. Wheelchair delays during this window cause permanent cognitive, language, and social deficits. Children without mobility cannot attend school, caregivers cannot work, and families remain trapped in poverty.

**Our goal**: Enable children to study, grow, become independent, and break the poverty trap.

## Success Criteria

A successful report will:
- Identify the most neglected age cohort with clear evidence
- Rank countries by urgency with supporting data
- Provide extractable data tables for presentations
- Note data gaps and confidence levels honestly
- Recommend 2-3 priority countries for first intervention

## Research Questions

### Primary
1. How many children (0-18) need wheelchairs in each ASEAN country?
2. Which age cohort (2-5, 6-12, 13-18) faces the worst gap in supply vs. need?
3. What do wheelchairs cost relative to household income in each country?
4. Why are children systematically underserved across ASEAN?

### Secondary
5. What government programs exist? What are their coverage gaps?
6. Which NGOs/Rotary clubs could be distribution partners?
7. What are the logistics for container import (ports, customs, duties)?

## Output Format

### 1. Executive Summary (300 words max)
- Most neglected age cohort + evidence
- Top 3 priority countries + rationale
- Key insight on why children are underserved

### 2. Age Cohort Analysis
Compare 2-5, 6-12, 13-18 across ASEAN. Which has worst gap? Least coverage? Highest intervention ROI? Recommend which to prioritize.

### 3. Country Profiles
One paragraph per country: gap size, wait times, economic barriers, partners, priority level.

### 4. Data Tables (CSV format)

**Table A: Wheelchair Gap by Country**
```csv
country,pop_under_18_millions,children_needing_wheelchair,children_without,gap_pct,avg_wait_months,confidence
```

**Table B: Age Cohort Gap**
```csv
country,need_2_5,gap_2_5_pct,need_6_12,gap_6_12_pct,need_13_18,gap_13_18_pct,most_neglected
```

**Table C: Economic Barriers**
```csv
country,b40_monthly_income_usd,wheelchair_retail_usd,cost_as_pct_income,confidence
```

**Table D: Infrastructure & Partners**
```csv
country,govt_program,coverage_pct,wheelchair_ngos,rotary_clubs,primary_port
```

**Table E: Priority Ranking**
```csv
country,gap_score,economic_score,logistics_score,partner_score,composite,rank
```

### 5. Recommendations
- Priority age cohort with rationale
- Phase 1 countries (immediate, one container)
- Phase 2 countries (6-12 months, need partner development)
- Key partners to engage

### 6. Data Confidence
Note which estimates are well-sourced versus extrapolated. Flag major data gaps. Use "low/medium/high" confidence ratings.

## Notes

- If data is unavailable for a country, say so rather than guessing
- Prioritize authoritative sources (WHO, UNICEF, national registries)
- Note conflicting sources when they exist
- Cross-reference estimates where possible
